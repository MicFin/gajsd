Intro to Angular 1.5.8

• Angular follows the MVC Architecture

• Use case scenarios for higher level functions:

⇒ Model: The Data (2-way binding)

⇒ View: Presents Data in HTML

⇒ Controller: Data Manipulation and User Interaction

• Additional Functionality:

⇒ Templating: similiar to Handlebars

⇒ Directives: special classes that do something, such as: ng-app, ng-controller, ng-repeat, ng-class

⇒ Factories & Services: modularized funtionality

Ceating Your First Angular App

• Adding an Angular module:


	var ToDoApp = angular.module("ToDoApp", []) { }

⇒ It takes in 2 params: 1) the app name, and, 2) dependencies which are added as an array, [] such as: ['ngAnimate', 'ui-router']

• Adding an Angular controller:


	ToDoApp.controller('ToDoAppCtrl', function($scope) { })

⇒ It takes in 2 params: 1) ctrl name, and, 2) a callback function that includes the required angular services

⇒ $scope is an object with properties and methods; it is accessible within our HTML templates

Adding Our App and Controller to HMTL

• Adding an angular module:


 	<div ng-app="ToDoApp"></div>

⇒ ng-app is an Angular directive, which is synonymous with a class or property; it auto-bootstraps the application scoping it to the <div> in which it was added

• Adding an angular controller:


 	<div id="wrapper" ng-controller="ToDoAppCtrl"></div>

⇒ ng-controller is another Angular directive and scopes the contorllers functionality to a specific section of the app

Adding Items To Our To Do List

• Grab input value using ng-model:


	<input type="text" id="new-item" ng-model="inputText">

• Adding a addItem method to our $scope:


	$scope.items = [];
  	$scope.addItem = function() {
	  $scope.items.push($scope.inputText)
	  $scope.inputText = ""
  	}

• Using ng-repeat to update the view with changes made to items array:


	<ul id="list">
	   <li ng-repeat="item in items">{{item}}</li>
	</ul>
Removing Items From Our To Do List

•Adding a removeItem method to our $scope:


  $scope.removeItem = function(index){ $scope.items.splice(index,1) }

• Adding an ng-click directive to the <li>:


  <li ng-repeat="item in items" ng-click="removeItem($index)">{{item}}</li>

• Using ng-repeat to once again update the view with changes made to items array:


  <ul id="list">
    <li ng-repeat="item in items" ng-click="removeItem($index)">{{item}}</li>
  </ul>
