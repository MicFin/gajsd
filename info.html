JavaScript: An Overview

What is JavaScript

• The language of the browser - aka the frontend

• Created in 10 days by Brendan Eich @ Netscape

• JavaScript ≠ Java

• One of the most popular programming languages *


	var greeting = 'Hello, World!'

	function hello (word) {
	  console.log(word)
	}

	hello(greeting)

	// "Hello, World!"
How is JavaScript Used?

• JavaScript is (almost) universal (write once, run everywhere)

• Frontend (client-side):

⇒ Used in the browser (alongside HTML and CSS)

⇒ Most common form

⇒ Included in, or referenced by, an HTML document

⇒ Designed to make web pages dynamic (vs. static)

•Backend (server-side):

⇒ Increasingly popular (to the ire of BE developers)

⇒ See NodeJS

•Mobile

⇒ See React Native
